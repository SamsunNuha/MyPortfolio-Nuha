# Samsun Nuha - Professional Portfolio

A modern, responsive single-page portfolio website showcasing professional skills, projects, and certifications. Built with the MERN stack in mind, using React for the frontend and Tailwind CSS for styling.

## 🚀 Live Demo
You can view the live site here: [http://localhost:3000](http://localhost:3000) (Local Development)

## 🛠️ Tech Stack
- **Frontend**: React.js (v19)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Components**: Swiper.js (for project carousels), React Icons
- **Deployment**: Prepared for platforms like Vercel or Netlify

## 📂 Project Structure
- `src/components/`: Modular UI components (About, Projects, Skills, Certifications, Header, Contact).
- `src/assets/`: Image and document assets (Certificates, Project screenshots, Logos).
- `tailwind.config.js`: Custom Tailwind theme configuration.
- `postcss.config.js`: PostCSS configuration for Tailwind.

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/SamsunNuha/my_portfolio.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
